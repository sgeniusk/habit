# Meet Recommendations And AI Memory

Updated: 2026-05-14

## 컨셉

Persona Habit의 모임은 사용자가 직접 방을 찾는 구조보다, 생활 스냅이 먼저 신호를 만들고 앱이 어울리는 모임을 제안하는 구조가 자연스럽다. 예를 들어 러닝 스냅이 반복되면 “성수천 러닝 모임 추천”을 띄우고, 도서관 공부가 반복되면 공부방을 제안한다.

리포트는 7일 요약만 보여주면 현재에 갇히기 쉽다. 그래서 오래된 스냅을 AI가 다시 꺼내 “그때의 나”를 되짚는 기억 모드가 필요하다. 이 기능은 단순 아카이브가 아니라, 페르소나의 시작점과 변화감을 발견하게 만드는 장치다.

## 현재 프로토타입

- `socialEngine`이 기록을 읽고 러닝, 도서관 공부, 식단 반복 신호를 모임 제안으로 변환한다.
- 모임 탭 상단에 `AI 모임 제안` 카드가 생겼다.
- 모임 제안은 `추천 고정`, `나중에 보기`, `관심 없음` 피드백으로 순서를 조정하거나 숨길 수 있다.
- 숨긴 모임 추천은 다시 보기 버튼으로 복구할 수 있고, 피드백 상태는 로컬 저장된다.
- 모임 제안의 CTA를 누르면 공유 가능한 초대 링크와 모임 대기실 카드가 생긴다.
- 초대 링크 복사, 공유 문구 준비, 초대 수락자 저장 상태를 확인할 수 있다.
- `/invite/:token` 경로로 들어오면 해당 모임 수락 화면이 먼저 열리고, 수락 후 대기실로 이어진다.
- 초대 수락 미리보기로 친구가 들어왔을 때의 참여 대기 상태와 공동 XP 증가를 확인할 수 있다.
- 대기실의 첫 스냅 미션을 완료하면 공동 페르소나 XP와 레벨이 오른다.
- 모임 멤버, 공동 XP, 첫 스냅 미션 완료 상태는 로컬 저장으로 유지된다.
- `memoryEngine`이 오래된 기록을 골라 월별 기억 카드, 요약, 회상 질문, 태그로 변환한다.
- 리포트 탭에 `7일 요약 / 오래된 기억` 전환 버튼을 추가했다.
- 오래된 기억은 월별, 장소별, 페르소나별 필터로 다시 탐색할 수 있다.
- AI 인사이트는 문구를 부드럽게 바꾸거나 관심 없는 항목을 숨기고 다시 복구할 수 있다.
- AI 인사이트 피드백은 새로고침 뒤에도 유지되도록 로컬 저장된다.
- 현재는 실제 AI 호출 없이 규칙 기반 큐레이션으로 제품 감각을 검증한다.

## 예시

반복 신호:

> 야외 운동 + 러닝 스티커 + 3km 기록이 2회 이상 쌓임

모임 제안:

> 성수천 러닝 모임 추천

초대 상태:

> 초대 링크 생성됨 -> 링크 복사됨 -> 친구 1명 참여 대기 -> +40 공동 XP -> 첫 스냅 완료 -> Lv.2 공동 러닝 페르소나

오래된 기억:

> 오래전의 러닝 감각

AI 회상 질문:

> 그날 뛰고 돌아온 뒤 무엇이 제일 시원했는지 떠올려볼까요?
