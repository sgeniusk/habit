# Persona Habit Alpha Report

Updated: 2026-05-15

## 보고

앞으로 이 프로젝트는 “아이디어 프로토타입”이 아니라 **알파 버전까지 내가 주도적으로 만들고 정리하는 제품 개발 프로젝트**로 관리한다.

알파 버전의 목표는 iOS/Android 출시 전이라도, 제3자가 웹 URL을 열어 다음 핵심 약속을 끝까지 체험하고 피드백할 수 있게 만드는 것이다.

> 생활 스냅을 남기면 페르소나가 자라고, AI가 내가 몰랐던 습관을 부드럽게 발견해 건강한 다음 행동으로 이어준다.

## 알파 기준

알파는 다음 조건을 만족해야 한다.

- 신규 사용자가 별도 설명 없이 `오늘 -> 스냅 -> 집 -> 모임 -> 리포트` 흐름을 이해한다.
- 생활 스냅은 사진, 필터, 스티커, 시간/횟수/페르소나 도장을 포함한다.
- 페르소나는 애칭, 직업, XP, 진화명, 방/의상 꾸미기 상태를 가진다.
- 오늘 탭은 날씨/위치 맥락과 한 줄 일기 작성 방식을 제공한다.
- 모임 탭은 기록 기반 추천, 초대 링크, 수락 화면, 첫 스냅 미션, 공동 XP를 제공한다.
- 리포트 탭은 7일 요약, 오래된 기억, AI 인사이트 피드백을 제공한다.
- 주요 상태는 새로고침 뒤에도 유지되어야 한다.
- GitHub Actions와 Vercel 배포가 항상 통과해야 한다.
- 문서는 PRD, 로드맵, 네이티브 전환 계획, 벤치마크, 알파 리포트를 포함해야 한다.

## 현재 알파 진행률

| 영역        | 상태    | 판단                                             |
| ----------- | ------- | ------------------------------------------------ |
| 스냅        | 구현 중 | 인증 도장과 기록 저장까지 들어왔고 export가 남음 |
| 페르소나 집 | 구현 중 | 애칭/직업/진화/꾸미기 저장까지 들어옴            |
| 오늘/일기   | 구현 중 | 한 줄 일기와 날씨 어댑터는 있음                  |
| 모임        | 구현 중 | 초대/미션/공동 XP는 있음                         |
| 리포트      | 구현 중 | 기억 탐색과 인사이트 피드백은 있음               |
| 저장        | 구현 중 | 모임/피드백/스냅/페르소나 설정 저장 완료         |
| 배포        | 통과    | GitHub, CI, Vercel 연결 완료                     |

현재 판단: **알파 0.75 수준**. 제품의 감정과 핵심 루프가 새로고침 뒤에도 유지되기 시작했고, 공유 가능한 결과물 내보내기와 첫 사용자 안내가 알파 완성의 가장 큰 빈칸이다.

## 알파까지 남은 핵심 작업

1. 스냅 기록 영속화
   - 완료: 새로고침 뒤에도 오늘 스냅, 최신 기록, 페르소나 XP가 유지된다.

2. 페르소나 설정 영속화
   - 완료: 애칭, 방 꾸미기, 의상 선택, 도장 선택을 저장한다.

3. 인증샷 결과물 강화
   - 페르소나 도장 위치를 선택할 수 있게 한다.
   - 공유용 이미지 저장/export 경계를 만든다.

4. 알파 온보딩
   - 첫 사용자가 앱의 핵심 루프를 30초 안에 이해할 수 있도록 첫 화면의 안내를 최소한으로 넣는다.

5. 알파 QA 체크리스트
   - 모바일 폭, 데스크톱 폭, 초대 링크, 새로고침, Vercel 배포, 접근성 이름을 점검한다.

## 자율 진행 원칙

- 사용자가 세부 지시를 주지 않아도 알파 기준에 맞춰 다음 작업을 선택한다.
- 새 기능은 테스트를 먼저 추가하고 실패를 확인한 뒤 구현한다.
- 기능이 늘어날수록 `src/lib`, `src/views`, `src/components`, `src/data` 경계를 유지한다.
- 매 개발 묶음마다 문서, 테스트, 빌드, 배포 상태를 함께 정리한다.
- 감성 기능은 과장하지 않고, 사용자가 실제로 계속 기록하고 싶어지는지로 판단한다.

## 완료한 개발 묶음

**상태 저장 알파 묶음**을 반영했다.

- 스냅 기록 localStorage 저장 완료
- 페르소나 애칭 저장 완료
- 방/의상 꾸미기 저장 완료
- 인증 도장 선택 저장 완료
- 새로고침 후 기록, 집, 스냅 화면 상태 복구 테스트 완료

## 다음 개발 묶음

바로 이어서 진행할 작업은 **공유용 인증샷 알파 묶음**이다.

- 페르소나 도장 위치 선택
- 인증샷 공유 이미지 export 경계
- 저장된 스냅에서 적용 도장/꾸미기 메타데이터 확인
- 첫 사용자 30초 온보딩 카드
- 모바일/데스크톱 시각 QA
