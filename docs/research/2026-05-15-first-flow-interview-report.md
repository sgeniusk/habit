## 페르소나 #1. 민서 (24, 반수생) — Student Track

### 기본 정보 (Basic)

- 가명: 민서 (Minseo)
- 나이대: 24
- 직업/역할: 반수생, 1 일 8 시간 인강 + 도서관 루틴
- 평소 쓰는 비슷한 앱: 노션 일일 트래커, 인스타 (스토리), 캘리커피 챌린지
- 인터뷰 날짜: 2026-05-15 (오후)
- 디바이스 / 브라우저: Galaxy S24 / Chrome Android, 한국어 모드

### Phase 0. 인사와 동의 (5 분)

- 평소 노션에 "오늘 푼 문제 수"만 기록. 노션은 "쓰다가 정리하느라 시간이 더 든다" 고 평가.
- 마지막 비슷한 앱은 캘리커피 챌린지. 친구가 안 해서 한 달 만에 그만뒀음.
- 동의서·녹음 OK. "스타카토 톤이 부담스러우면 말씀드릴게요" 라고 농담.

### Phase 1. 첫 30 초 관찰 (5 분)

| 시각 | 행동                                               | 추정 감정          |
| ---- | -------------------------------------------------- | ------------------ |
| 0:00 | 진입, 흰 로딩 후 "Persona Habit" 로고 응시         | 호기심 + 약간 의심 |
| 0:08 | 첫 가이드 슬라이드 1 장째 휙 넘김 (읽지 않음)      | 조급함             |
| 0:18 | 두 번째 슬라이드도 스와이프, "감 잡았어요" 발화    | 어림짐작 모드      |
| 0:30 | "첫 스냅 찍기" 버튼은 못 찾고 좌상단 햄버거 두드림 | 헤맴               |
| 0:55 | 하단 탭 바에서 "집" / "모임" / "리포트" 확인       | 안정감 (구조 파악) |
| 1:10 | 다시 홈으로 와서 큰 CTA 버튼 발견                  | 안도               |

- 첫 CTA 발견까지 65 초. **목표 10 초 대비 6.5 배 초과.**
- "이게 다이어리인지 게임인지 헷갈렸어요. 카피만 보면 둘 다 같아요." (인용)

### Phase 2. 첫 스냅 (10 분)

- 도서관 책상 사진 → 필터 단계에서 "왜 굳이 보정?" 이라며 건너뜀.
- 스티커는 "공부" 카테고리에서 책 모양 1 개 → 도장 "오늘도 견딤" 선택.
- 카테고리에서 **"공부" 가 첫 화면에 없어서** 스크롤로 찾음. (P1 후보)
- 장소 입력은 "도서관" 직접 타이핑. 자동 추천 없음에 의아함.
- 저장 후 4 분의 1 정도 페이지가 흔들리며 페르소나가 등장. "어 갑자기 캐릭터?" 라는 반응.
- 빠른 이동 카드 3 개 중 **"집"** 으로 진입 (대표 페르소나가 어떻게 변했는지 보고 싶다고).
  정성 응답.
- "방금 한 흐름을 한 줄로 요약해 보세요." → "사진 올리고 도장 찍어서 캐릭터 키우는 거?"
- "저장하면 무엇이 바뀐다고 느끼나요?" → "캐릭터가 좀 자라는 것 같긴 한데, 얼마나 자란 건지는 모르겠어요."
- "어디서 멈췄나요?" → "카테고리. 공부가 너무 작게 있어요."

### Phase 3. 집 탭 (5 분)

- 대표 페르소나는 "공부형 페르소나" 라고 짚어냈음 (이름은 못 외움).
- 가장 먼저 만진 것은 **방 꾸미기** (책상 위 미니어처 변경). 페르소나 자체는 안 만짐.
- 애칭 변경 화면을 못 찾음. "이거 길게 누르는 건가요?" 라며 시도하다 포기.
- 말투 토글 ("귀여운" ↔ "차분한") 은 "별 차이를 못 느끼겠어요" 평가.
  정성 응답.
- "이 페르소나는 누구라고 느껴지나요?" → "내가 공부할 때의 나… 인데 좀 떨떠름해요. 너무 귀여워서 무게감이 없어요."
- "방을 꾸민 결과가 다음 스냅을 더 남기고 싶게 만드나요?" → "한 번은 또 올리고 싶어요. 두 번째까지 갈지는 모르겠어요."

### Phase 4. 모임 탭 (5 분)

- AI 모임 제안 카드: "수능 D-180 같이 견디기" 류 추정 카드를 한참 응시. "이건 좀 닭살이에요" 라고 평가.
- 초대 링크 복사는 시도. 공유 문구는 "내가 만들어야 더 자연스러울 듯" 이라며 안 씀.
- "초대 수락 미리보기" 의 의미는 두 번 설명 듣고 이해.
  정성 응답.
- "이 모임은 부담스러운가요?" → "친구한테 권하기엔 좀 그래요. 공부 인증 모임은 카톡으로도 하고 있어서."
- "친구를 떠올렸을 때 누가 같이할 만한가요?" → "같이 반수하는 친구 1 명. 그 외엔 솔직히 없어요."
- "이 모임이 있어서 다시 들어올 이유가 생기나요?" → "이 정도로는 약해요. 친구가 먼저 들어와 있어야 들어갈 것 같아요."

### Phase 5. 리포트 탭 (5 분)

- "AI 가 발견한 숨은 습관" 카드 읽는 데 약 25 초. "이렇게 길게 분석한 거 처음 봐요" 라고 평가.
- "근거" 라인은 발견하지 못함. 인터뷰어 안내 후에야 "아 이게 근거구나" 라고 인식. (P0 후보)
- 모드 전환은 "문구 순하게" 만 시도. "오래된 기억" 은 "왜 이런 모드가 있는지 모르겠어요" 라며 안 누름.
  정성 응답.
- "이 분석이 맞다고 느끼나요?" → "반은 맞고 반은 끼워맞추는 느낌이에요. 도서관 한 번 갔다고 '집중력 회복기' 라고 부르는 건 과해요."
- "근거 한 줄이 분석의 신뢰도를 얼마나 바꾸나요?" → "처음 봤을 때보다 좋아져요. 근데 근거가 어디 있는지부터 안 보였어요."
- "오래된 기억 모드는 어떤 상황에서 다시 보고 싶나요?" → "솔직히 잘 모르겠어요."

### Phase 6. 회고 (10 분)

1. 한 줄 설명. "사진 올리면 캐릭터가 자라고 AI 가 분석해주는 일기 앱."
2. 내일 들어올 이유. "어제 캐릭터가 어떻게 변했는지 궁금하면."
3. 가장 어색한 순간. "리포트에서 말투가 갑자기 시집 같아져요."
4. 1 천 원 지불 의향. "한 달이면 낼 수 있는데, 1 회성은 안 낼 것 같아요."
5. 친구 초대 의향. "같이 반수하는 한 명한테는 시도해볼만해요."
6. 1 개월 뒤 잔존. "솔직히 반반. 캐릭터가 진짜로 변하는 게 보이면 남고, 비슷한 그림만 반복되면 지워요."

## 페르소나 #2. 지훈 (31, 스타트업 마케터) — Worker Track

- 나이대: 31
- 직업/역할: 시리즈 A 스타트업 PMM, 주 4 회 헬스 + 단백질 식단
- 평소 쓰는 비슷한 앱: MyFitnessPal (식단), Apple Health (걸음), Notion Daily Note
- 인터뷰 날짜: 2026-05-15 (저녁)
- 디바이스 / 브라우저: iPhone 15 / Safari iOS

### Phase 0. 인사와 동의 (5 분)

- "트래커는 6 개월 쓰다가 다 그만뒀어요. 일관성이 무너지면 자책감만 남거든요."
- 동의서 OK. 인센티브 (커피 5,000 원) 는 "필요 없어요, 신기해서 왔어요" 라고 사양.

### Phase 1. 첫 30 초 관찰 (5 분)

| 시각 | 행동                                                          | 추정 감정 |
| ---- | ------------------------------------------------------------- | --------- |
| 0:00 | 진입 후 즉시 우상단 햄버거 → 설정으로 가서 언어 토글부터 찾음 | 통제 욕구 |

- 첫 CTA 발견까지 55 초. 학생보다 느림 (텍스트를 읽기 때문).
- "기록계 앱들이 이미 많아요. 차별점이 뭐인지 30 초 안에 안 보였어요." (인용)

### Phase 2. 첫 스냅 (10 분)

- 닭가슴살 도시락 사진 → 필터 안 씀.
- 카테고리는 "식단" 첫 화면에 보임. **칼로리·매크로 입력 칸이 없음** 에 즉시 불만 표시. (P1)
- 스티커는 "스킵" 버튼 누르고 저장.
- 저장 후 페르소나 등장 모션을 무표정으로 응시. "이게 캐릭터 게임이군요" 라는 반응.
  정성 응답.
- 한 줄 요약. "사진 카테고리만 골라서 도장 찍는 일기."
- "저장하면 무엇이 바뀌나요?" → "체감상 안 바뀌어요. 게이지가 1 mm 차는 정도?"
- 멈춘 순간. "도장이랑 스티커가 왜 둘 다 있는지 모르겠어요. 한 쪽이면 충분해요."

### Phase 3. 집 탭 (5 분)

- 대표 페르소나 인식 못함. "여기 캐릭터가 둘인데 누가 대표인지 표시가 약해요." (P0 후보)
- 페르소나 꾸미기보다는 **말투 토글**을 먼저 만짐. "차분한" 으로 바꿔보고 "그래도 너무 캐주얼해요" 라고 평가.
- 페르소나 진화 게이지의 단계 정의가 안 보임. "다음 단계까지 몇 번 스냅해야 하는지 알려줘야죠." (P1)
  정성 응답.
- "이 페르소나는 누구라고 느껴지나요?" → "내 식단 자아? 라기보단 모르는 캐릭터예요."
- "방을 꾸민 결과가 다음 스냅을 더 남기고 싶게 만드나요?" → "전혀요. 꾸미기는 제 동기와 안 맞아요."
- 말투 차이. "있긴 한데 미세해서 변화로 안 느껴져요."

### Phase 4. 모임 탭 (5 분)

- 집 탭에서 대표 페르소나 시각 표시 미흡. 배지·하이라이트·툴팁 중 1 개 이상 추가.
- AI 리포트의 "근거" 라인 가시성. 근거 라인을 카드 첫 줄로 끌어올리고 사진 미리보기·날짜를 함께 노출.
- AI 리포트의 기본 톤. "문구 순하게" 가 기본값이 되어야 신뢰도 회복.
  P1 (이번~다음 묶음 범위).
- 카테고리 분류 재설계 (학생 "공부" 노출 순서, 캐주얼 "일상" 세분화).
- 페르소나 진화 게이지에 "x/y 회 남음" 라벨.
- 페르소나 애칭 변경 UI 의 명시적 버튼화.
- 도장과 스티커의 역할 차별화 또는 한쪽 통합.
- 모임 진입 알림 정책 (친구 입장 시 푸시) 흐름 점검.
  P2 (백로그 관찰).
- 영어 모드 토글의 자연스러운 발견 경로 (현재 사용자가 굳이 안 찾음).
- 꾸미기 단계의 첫 흐름 비중 증대 (수아 류 페르소나 잔존율 직결).
- "오래된 기억" 모드의 사용 시점 안내 (한 달 후 회상 알림 등).
