## Persona Habit에 적용한 방향

Persona Habit의 인증샷은 “증거 사진”이면서 동시에 “페르소나와 같이 찍은 성장 카드”가 되어야 한다. 그래서 일반적인 필터/스티커 위에 다음 도장을 추가했다.

- 시간 도장: 지금 남긴 스냅이라는 감각을 만든다.
- 횟수 도장: 오늘 몇 번째 기록인지 보여준다.
- 페르소나 도장: 사용자가 정한 애칭과 직업을 사진 구석에 붙인다.

예시:

```text
23:50 인증
오늘 1회차
곰곰이와 함께해요
직업 · 학습자
```

## 애칭과 직업

페르소나는 “직업/역할”과 “애칭”을 분리한다.

- 직업: 학습자, 러너, 식단러처럼 습관 카테고리를 설명한다.
- 애칭: 곰곰, 토리처럼 사용자가 부르는 이름이다.

공부할 때의 나는 “학습자 곰곰이”가 되고, AI 대화는 “곰곰아. 이번에는 공부를 많이 했네.”처럼 애칭을 기준으로 말한다.

## 진화 농담

학습 XP가 충분히 쌓이면 `척척박사 페르소나`로 진화한다. 너무 진지한 건강/습관 앱보다, 사용자가 피식 웃으면서 계속 찍게 만드는 보상 언어를 의도한다.

## 다음 연결

- 시간 도장을 실제 촬영 시점으로 고정 저장
- 위치/날씨 도장을 선택 옵션으로 추가
- 페르소나 도장 위치를 네 귀퉁이 중 선택
- 공유 이미지 export 또는 저장 기능
- 각 페르소나별 애칭 저장소를 서버 계정과 동기화
