# Persona One-Line Journal

Updated: 2026-05-14

## 컨셉

오늘의 일기는 긴 글을 쓰는 화면이 아니라, 페르소나에게 한 줄씩 말을 건네는 대화 화면이다. 사용자는 “오늘은 괜히 멀리 걷고 싶었어”처럼 툭 던지고, 페르소나는 날씨, 습도, 위치, 이동 거리 같은 맥락을 섞어 감정과 분위기를 정리해준다.

핵심 감각은 펫처럼 곁에 있지만, 실제로는 사용자의 내면 자아를 키우는 존재에 가깝다.

## 현재 프로토타입

- 오늘 탭의 `AI랑 같이쓰기 / 혼자 기록하기`에서 일기 모드를 선택한다.
- AI 모드는 날씨, 습도, 집에서 떨어진 거리로 먼저 말을 건다.
- 위치/날씨 카드는 허용, 거부, 실패 상태를 전환하며 문구와 행동 버튼이 바뀐다.
- `weatherEngine`이 실제 날씨 API 연결 전 사용할 어댑터 인터페이스와 화면 상태 모델을 제공한다.
- 사용자가 한 줄을 입력하면 원문, 페르소나 응답, 정리된 한 줄, 감정 태그가 생긴다.
- 현재는 실제 모델 호출 없이 `journalEngine`의 규칙 기반 응답으로 감각을 검증한다.

## 대화 예시

사용자:

> 오늘은 괜히 멀리 걷고 싶었어

페르소나:

> 집에서 조금 멀리 나왔네. 어디 좋은 데 놀러 가? 돌아오는 길의 느낌도 나한테 말해줘.

정리된 한 줄:

> 맑은 날씨에 멀리 걸으며 마음의 방향을 다시 잡은 날.
